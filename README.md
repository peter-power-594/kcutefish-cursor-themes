# Cutefish Cursor Themes

forked from https://github.com/ful1e5/Bibata_Cursor

## OpenMandriva Dependencies

```shell
sudo dnf in task-develop
```

## Build

```shell
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=$(YOUR_DIR) ..
make
```

## Install

```shell
sudo make install
```
